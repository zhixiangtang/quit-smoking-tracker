# 戒烟日期记录 🚭

一个简单易用的戒烟日期记录Web应用，帮助用户追踪戒烟天数、节省金额和健康改善。

## 功能特性

✅ **核心功能**
- 设置戒烟开始日期
- 实时计算戒烟天数
- 显示节省金额
- 健康评分系统

✅ **用户体验**
- 响应式设计，支持手机和电脑
- 本地存储，数据持久化
- 流畅的动画效果
- 离线可用（PWA）

✅ **额外功能**
- 戒烟里程碑展示
- 分享成就到社交媒体
- 生成分享图片
- 多种时间单位显示

## 使用方法

### 在线访问
直接访问部署在 GitHub Pages 的地址：https://[你的用户名].github.io/quit-smoking-tracker/

### 本地运行
1. 克隆仓库
2. 使用浏览器直接打开 `index.html`
3. 无需安装任何依赖

## 部署到 GitHub Pages

1. Fork 或克隆此仓库
2. 在仓库设置中开启 GitHub Pages
3. 选择部署分支（通常是 main）
4. 访问 `https://[你的用户名].github.io/quit-smoking-tracker/`

## 技术栈

- HTML5 / CSS3 / JavaScript
- 本地存储 API
- Canvas API（分享图片生成）
- PWA 技术

## 数据存储

所有数据都保存在用户的浏览器本地存储中：
- `quitDate`: 戒烟开始日期
- `dailyCost`: 每日吸烟花费
- `displayUnit`: 显示单位设置

## 浏览器支持

支持所有现代浏览器：
- Chrome 60+
- Firefox 55+
- Safari 11+
- Edge 79+
